MongoDB Scanner
---------------
This a basic MongoDB scanner written in Ruby.

Requirements
------------
* Ruby 3.2.2
* Bindata
* BSON
* OpenSSL

How to install the dependencies
-------------------------------
```
bundle install
```

Usage
-----
```
bin/mongodb_scanner -i <ip> -p <port>
```

or

```
bin/mongodb_scanner --help
```
